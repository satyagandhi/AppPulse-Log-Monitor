# AppPulse – Application Log Monitoring Tool

AppPulse is a simple Python-based log monitoring project that simulates basic application support tasks.

## Features
- Reads application log files
- Detects ERROR and WARNING messages
- Counts system issues
- Generates simple alerts

## Technologies
Python  
Log Monitoring  
Application Support Concepts

## Purpose
This project demonstrates basic application support skills such as log monitoring, issue detection, and operational troubleshooting.
